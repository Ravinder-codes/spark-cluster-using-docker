# spark-cluster-using-docker
Creating a simple spark cluster with the help of docker containers. This repo is for the visualization purpose, learning how spark works in distributed nodes.

```Install Docker on your desktop/laptop```<br>
You can easily install it on linux<br>
For windows, use Docker Desktop

```Copy the .env_template.txt file```<br>
Copy the file and create .env file<br>
Upate the configurations as per your need

```Run the spark cluster```<br>
Run the command >>> docker-compose up<br>
The program will automatically run the spark.py file in the app folder

```Stop the spark cluster```<br>
Run the command >>> docker-compose down
